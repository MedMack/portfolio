Prototyping - Postcard App

A freelance contract to prototype and design a postcard mobile app project on upwork

The project

A client was looking for someone to map out the UX and then design the UI of a mobile application which allow users to send postcards in the real world.

The project was initially a web platform, which needed a mobile version to reach more users as it's the most used platform nowadays.

The app

The mobile application was not that big, but the deadline to deliver was not much. The mobile app will have the following features :

  • Login
  • Sign-up
  • Forgot password process
  • Design a postcard (front and back)
  • Upload and edit images
  • Address a postcard (select senders)
  • Send and payment
  • Address book management
